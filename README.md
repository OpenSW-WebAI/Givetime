# GIVETIME - 재능기부 매칭 플랫폼

GIVETIME은 시간과 재능을 나누고 싶은 사람과, 도움이 필요한 사람을 연결해주는 재능기부 플랫폼입니다.  
쉽고 직관적인 UI/UX, 봉사시간 자동 기록 시스템, 감동적인 이야기 공유 등으로 지속 가능한 선순환을 만들어갑니다.

## 주요 기능

- 재능 등록 / 요청  
  사용자는 자신의 재능을 등록하거나, 필요한 도움을 요청할 수 있습니다.

- 봉사시간 자동 기록 
  기부 활동이 완료되면 봉사 시간이 자동으로 기록되어 추후 증빙이 가능합니다.

- 랭킹 및 레벨 시스템  
  누적된 봉사시간에 따라 사용자 랭킹이 매겨지고, 활동 이력에 따른 레벨업 시스템이 적용됩니다.

- 따뜻한 이야기 게시판  
  실제 기부자와 수혜자의 감동적인 후기를 공유하여 커뮤니티 정서적 유대를 강화합니다.

## 주요 페이지

| 페이지 | 주요 내용 |
|--------|-----------|
| 메인 페이지 | 모집 중인 재능 기부 목록, 따뜻한 이야기 슬라이드 배너 |
| 로그인 / 회원가입 | 모달 형태, 소셜 로그인 (Google), 자동 로그인 기능 |
| 재능 등록 / 요청 | 카테고리 필터, 인기글, 활동 상태 표시 |
| 프로필 페이지 | 봉사시간 기록, 레벨 시스템, 멘토 연락 기능 제공 |

## 기대 효과

- 커뮤니티 강화
  누구나 쉽게 기부할 수 있는 구조로 사회적 유대감을 향상

- 봉사문화 확산  
  실생활 속에서 자연스럽게 봉사를 실천할 수 있는 시스템 제공

- 개인 성장 기회  
  기부자 스스로의 자신감과 실전 경험 축적
